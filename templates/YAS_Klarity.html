LTD FAST YAS / Klarity Common Issues

Inspec tools:
- Lot finished stuck in RUN and has message "Poling for yas data file"


Review Tools:
- User ranlot on Review tool but not seeing images in Klarity

- User gets popup message "The maximum time to wait for a tool spc file has been reached"


Classification Tools:
- Lot aborts on DOR tool due to "Lot found in ICE Server, but no image data available"

- Lot aborted on DOR tool due to "Lot Not Found in ICE Server, No data available"

- Lot finished classifying on DOR tool but getting message "The maximum time to wait for a YAS Sfile has been reached. Select OK to repeat the wait loop or Cancel to Abort the Lot".     -    D1C Example of same issue


General YAS / Klarity issues:
